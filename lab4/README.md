Лабораторная работа №4

Описание
Десктопное приложение для управления с графическим интерфейсом на Windows Forms. Портирование консольного приложения в Windows Forms.


Структура проекта
MainForm.h - Главная форма приложения
MainForm.cpp - Реализация главной формы
AddEditForm.h - Форма добавления/редактирования номеров
AddEditForm.cpp - Реализация формы редактирования
HotelClasses.h - Классы Room и Hotel для хранения данных
HotelClasses.cpp - Реализация классов данных


Функционал
Главное окно управления номерами гостиницы
Второе окно для добавления/редактирования номеров
CRUD операции: добавление, удаление, изменение данных номеров
Валидация ввода данных (проверка типов, диапазонов)
Сортировка номеров по стоимости
Сохранение и загрузка данных из файла
Отображение данных в DataGridView
Статистика - средняя стоимость и количество номеров
Перегрузка конструкторов форм для разных сценариев


Особенности реализации
Разделение на файлы - классы вынесены в отдельные файлы
Два окна с переходами между ними
Перегрузка конструкторов форм для режимов добавления/редактирования
Проверки корректности вводимых данных
Работа с файлами для сохранения состояния
DataGridView для отображения данных
OpenFileDialog/SaveFileDialog для работы с файлами
