Лабораторная работа №5
Управление гостиничными номерами с БД

Описание
Десктопное приложение на Windows Forms (C++/CLI) для управления номерами гостиницы с подключением к локальной БД SQLite.

Функционал
- Главное окно и окно редактирования
- Подключение к SQLite базе данных
- Сохранение данных из БД в файл
- Загрузка данных из файла в БД 
- Добавление, изменение, удаление записей
- Валидация ввода
- DataGridView для отображения
- OpenFileDialog для работы с файлами
- Отдельные файлы для классов (.h/.cpp)

Структура проекта
MainForm.h/cpp
AddEditForm.h/cpp
HotelClasses.h/cpp
DatabaseManager.h/cpp
HotelDatabase.db
